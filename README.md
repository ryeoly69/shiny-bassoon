# A PARIS BAGEL GALLERY Manual

This repository includes a single HTML file `index.html` that contains the store manual.

## Viewing

Open the file directly in your browser or run a simple HTTP server from the repository root:

```bash
python3 -m http.server 8000
```

Then visit [http://localhost:8000/index.html](http://localhost:8000/index.html) in your browser.
